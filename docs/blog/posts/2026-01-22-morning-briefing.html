Good Morning, Captain: Proactive Morning Briefings

The Officer of the Watch now greets you each morning with everything you need to know.

Previously: Check calendar. Check tasks. Check alerts. All separate. Now: One briefing, all the context.

The Problem

Morning routine: Open calendar, scan events. Open task manager, find urgent items. Check email, look for overnight fires. Repeat across multiple tools, multiple contexts.

By the time you're done gathering context, you've lost valuable minutes. And you might have missed something.

The Solution: Morning Briefing

The Officer of the Watch generates a proactive briefing every morning:

from klabautermann.agents.officer import OfficerOfTheWatch, OfficerConfig

officer = OfficerOfTheWatch(
    neo4j_client=neo4j,
    config=OfficerConfig(
        briefing_hour=7,      # Generate at 7:00 AM
        briefing_minute=0,
    ),
)

briefing = await officer.generate_morning_briefing()
print(briefing.greeting)
# "Good morning, Captain. Moderate winds - a full but manageable schedule."

print(briefing.summary_text)
# **Schedule**: 4 event(s) today
#   - 09:00: Team Standup at Zoom
#   - 11:00: Product Review at Room A
#   - 14:00: 1:1 with Sarah
#   - 16:00: Sprint Planning
#
# **High Priority Tasks**: 2 task(s)
#   - [!] Deploy hotfix (Backend)
#   - [H] Review security audit
#
# **Overdue**: All clear

What's in a Briefing

The MorningBriefing includes everything you need:

  • events_today: Calendar events for the day, sorted by start time
  • high_priority_tasks: Urgent and high-priority tasks
  • overdue_tasks: Tasks past their due date
  • overnight_alerts: Deadline warnings and overdue alerts
  • greeting: Nautical-themed greeting based on workload
  • summary_text: Human-readable markdown summary

Greetings Based on Outlook

The greeting adapts to your day's workload:

Workload Greeting
No events or tasks "Clear skies ahead - no events or urgent tasks today."
1-3 items "Calm seas - a manageable day ahead."
4-6 items "Moderate winds - a full but manageable schedule."
7+ items "Choppy waters - a busy day ahead. Steady as she goes."
Overdue tasks "There are overdue items requiring attention."

Configuration

Customize the briefing time to match your schedule:

config = OfficerConfig(
    briefing_hour=6,       # Early bird? 6:00 AM
    briefing_minute=30,    # Half past the hour
    briefing_look_ahead_days=1,  # Days to look ahead
)

Scheduler Integration

Check if it's briefing time with the helper method:

if officer.is_briefing_time():
    briefing = await officer.generate_morning_briefing()
    # Send to all channels
    await channel_manager.broadcast(briefing.summary_text)

Process Message Interface

Request a briefing through the standard agent interface:

msg = AgentMessage(
    source_agent="orchestrator",
    target_agent="officer_of_the_watch",
    intent="briefing",
    payload={"operation": "morning_briefing"},
    trace_id=trace_id,
)

response = await officer.process_message(msg)
briefing_data = response.payload
