Numbers That Matter

Today we added Prometheus metrics to Klabautermann. Because knowing your ship is running smoothly is different from being able to prove it.

The Observability Triad

With structured logging (#272) and workflow inspection (#357) already aboard, we now complete the observability triad:

  • Logs - What happened and when (JSON-structured)
  • Traces - How requests flow through agents (workflow inspector)
  • Metrics - How the system is performing (Prometheus)

Each serves a different purpose. Logs for debugging specific events. Traces for understanding agent interactions. Metrics for dashboards, alerts, and trends.

The Implementation

A custom registry for clean separation:

from prometheus_client import CollectorRegistry, Counter, Histogram, Gauge

REGISTRY = CollectorRegistry()

AGENT_REQUESTS_TOTAL = Counter(
    "klabautermann_agent_requests_total",
    "Total requests processed by agents",
    ["agent_name"],
    registry=REGISTRY,
)

Helper functions for clean integration:

def record_agent_request(agent_name: str) -> None:
    AGENT_REQUESTS_TOTAL.labels(agent_name=agent_name).inc()

def record_agent_latency(agent_name: str, latency_ms: float) -> None:
    AGENT_REQUEST_LATENCY_MS.labels(agent_name=agent_name).observe(latency_ms)

BaseAgent integration is automatic:

async def _handle_message(self, msg: AgentMessage) -> None:
    start_time = time.time()
    record_agent_request(self.name)
    set_agent_inbox_size(self.name, self.inbox.qsize())

    try:
        response = await self.process_message(msg)
        record_agent_success(self.name)
    except Exception:
        record_agent_error(self.name)
        raise
    finally:
        elapsed_ms = (time.time() - start_time) * 1000
        record_agent_latency(self.name, elapsed_ms)

The Endpoint

Metrics are exposed at /metrics:

@app.get("/metrics")
async def metrics_endpoint() -> Response:
    return Response(
        content=get_metrics(),
        media_type="text/plain; version=0.0.4; charset=utf-8",
    )

Standard Prometheus text format. Scrape with any Prometheus instance.

API Middleware

Request timing happens automatically:

@app.middleware("http")
async def metrics_middleware(request: Request, call_next):
    start_time = time.perf_counter()
    response = await call_next(request)
    elapsed = time.perf_counter() - start_time

    if request.url.path != "/metrics":  # Avoid recursion
        record_api_request(request.method, request.url.path, response.status_code)
        record_api_latency(request.method, request.url.path, elapsed)

    return response

Timed Operations

For custom timing, there's a decorator:

from klabautermann.core.metrics import timed_operation, record_graph_latency

@timed_operation(record_graph_latency, "search")
async def search(self, query: str) -> list[dict]:
    # Latency automatically recorded
    return await self._execute_search(query)

Works with both sync and async functions. Records even when exceptions occur.
