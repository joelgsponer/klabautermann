Scraping the Hull

Knowledge graphs grow. Relationships multiply. Some connections fade into irrelevance.

The Hull Cleaner keeps things tidy.

The Problem

Over time, relationship weights decay. A connection mentioned once three months ago might now have a weight of 0.15. It clutters search results. It slows traversals. It needs to go.

The Solution

from klabautermann.agents import HullCleaner, HullCleanerConfig

config = HullCleanerConfig(
    weak_relationship_threshold=0.2,  # Weight below this is "weak"
    weak_relationship_age_days=90,    # Must be at least this old
    dry_run_by_default=True,          # Preview before deleting
)

cleaner = HullCleaner(neo4j_client=neo4j, config=config)

Finding Weak Relationships

First, see what's eligible for pruning:

weak_rels = await cleaner.find_weak_relationships()

for rel in weak_rels:
    print(f"{rel.source_name} -[{rel.relationship_type}]-> {rel.target_name}")
    print(f"  Weight: {rel.current_weight:.3f}")
    print(f"  Days since access: {rel.days_since_access:.0f}")

Dry Run Mode

Never delete blindly. Preview first:

# Preview what would be deleted
result = await cleaner.prune_weak_relationships(dry_run=True)

print(f"Found {result.relationships_found} weak relationships")
print(f"Would prune {result.relationships_pruned}")

# Review the audit entries
for entry in result.audit_entries:
    print(f"  {entry.action}: {entry.reason}")

Actual Pruning

When you're ready:

# Actually delete weak relationships
result = await cleaner.prune_weak_relationships(dry_run=False)

print(f"Pruned {result.relationships_pruned} relationships")
print(f"Errors: {result.errors}")

Full Maintenance

Run all cleanup tasks at once:

result = await cleaner.scrape_barnacles(dry_run=True)

print(f"Total relationships found: {result.relationships_found}")
print(f"Total to prune: {result.relationships_pruned}")
print(f"Duration: {result.duration_ms:.1f}ms")

Graph Health Statistics

Monitor your graph's condition:

stats = await cleaner.get_pruning_statistics()

print(f"Total relationships: {stats['total_relationships']}")
print(f"Average weight: {stats['avg_weight']:.3f}")
print(f"Weak count: {stats['weak_count']}")
print(f"Weak percentage: {stats['weak_percentage']:.1f}%")

Audit Trail

Every operation is logged:

# Get current session's audit log
log = cleaner.get_audit_log()

for entry in log:
    print(f"{entry['timestamp']} - {entry['action']}")
    print(f"  Entity: {entry['entity_type']} {entry['entity_id']}")
    print(f"  Reason: {entry['reason']}")

# Clear when done
cleaner.clear_audit_log()

Configuration

Tune the thresholds:

config = HullCleanerConfig(
    # Enable/disable pruning rules
    prune_weak_relationships=True,

    # Weak relationship criteria
    weak_relationship_threshold=0.2,  # Weight threshold
    weak_relationship_age_days=90,    # Minimum age in days

    # Safety limits
    max_deletions_per_run=1000,
    dry_run_by_default=True,

    # Scheduling (for future automation)
    schedule_cron="0 2 * * 0",  # Sunday 02:00
)
