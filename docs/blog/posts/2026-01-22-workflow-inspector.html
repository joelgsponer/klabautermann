The Black Box Opens

Multi-agent systems are powerful but opaque. When the Researcher returns "I found nothing," was it a bad query? A search timeout? An empty graph? When the Executor fails, was the action invalid or did MCP timeout? Today we added a window into each agent's mind.

The Problem

In a multi-agent system, debugging means tracking messages across multiple async processes. Standard logging helps, but what you really need is:

  • What did the agent receive? (REQUEST)
  • What decisions did it make? (THINKING)
  • What did it produce? (OUTPUT)

The existing logger shows events as they happen. The workflow inspector shows the complete lifecycle of each agent's work, organized by trace ID.

The Inspector

WorkflowInspector captures three phases for every agent interaction:

class WorkflowPhase(str, Enum):
    REQUEST = "REQUEST"   # What came in
    THINKING = "THINKING" # What the agent decided
    OUTPUT = "OUTPUT"     # What went out

Automatic Logging

REQUEST and OUTPUT phases are logged automatically by BaseAgent._handle_message():

async def _handle_message(self, msg: AgentMessage) -> None:
    inspector = get_inspector()

    # Log what came in
    inspector.log_request(
        trace_id=msg.trace_id,
        agent_name=self.name,
        data={
            "intent": msg.intent,
            "source": msg.source_agent,
            "payload": msg.payload,
        },
    )

    response = await self.process_message(msg)

    # Log what went out
    inspector.log_output(
        trace_id=msg.trace_id,
        agent_name=self.name,
        data={
            "status": "success",
            "response_payload": response.payload if response else None,
        },
        duration_ms=elapsed_ms,
    )

Every agent inherits this behavior. No extra code needed.

THINKING Phase

The interesting part is what happens between REQUEST and OUTPUT. Each agent logs its decisions:

Researcher

# After planning
log_thinking(
    trace_id=trace_id,
    agent_name=self.name,
    data={
        "step": "plan_ready",
        "reasoning": plan.reasoning,
        "strategies": [s.technique.value for s in plan.strategies],
        "expected_result_type": plan.expected_result_type,
    },
)

# After search execution
log_thinking(
    trace_id=trace_id,
    agent_name=self.name,
    data={
        "step": "search_complete",
        "raw_results_by_technique": {k: len(v) for k, v in results.items()},
        "aggregated_count": len(aggregated),
    },
)

Ingestor

# After text cleaning
log_thinking(
    trace_id=msg.trace_id,
    agent_name=self.name,
    data={
        "step": "text_cleaning",
        "original_length": len(text),
        "cleaned_length": len(cleaned),
        "removed_chars": len(text) - len(cleaned),
    },
)

Executor

# After parsing action
log_thinking(
    trace_id=msg.trace_id,
    agent_name=self.name,
    data={
        "step": "parsed",
        "action_type": request.type.value,
        "target": request.target,

# After validation
log_thinking(
    trace_id=msg.trace_id,

File Output

For analysis, entries write to logs/workflow_inspection.jsonl:

{"trace_id":"abc12345","agent":"researcher","phase":"REQUEST",...}
{"trace_id":"abc12345","agent":"researcher","phase":"THINKING",...}
{"trace_id":"abc12345","agent":"researcher","phase":"OUTPUT",...}

JSONL format makes it easy to grep, filter, and analyze with standard tools.

Configuration

# Enable inspection (default: enabled)
export WORKFLOW_INSPECT=true

# Filter to specific agents
export WORKFLOW_FILTER_AGENTS=researcher,executor

# Disable console output (keep file output)
export WORKFLOW_CONSOLE=false

# Custom log file path
export WORKFLOW_LOG_FILE=/var/log/klabautermann/workflows.jsonl

In-Memory Buffer

For testing and programmatic access:

inspector = get_inspector()

# Get all entries for a trace
entries = inspector.get_entries(trace_id="abc12345")

# Get summary
summary = inspector.get_workflow_summary("abc12345")
# {"trace_id": "abc12345", "agents": {"researcher": {"phases": ["REQUEST", "THINKING", "OUTPUT"], "total_duration_ms": 1444.2}}}

# Filter by phase
thinking_entries = inspector.get_entries(phase=WorkflowPhase.THINKING)
