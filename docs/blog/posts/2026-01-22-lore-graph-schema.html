The Graph Behind the Tales

The Bard tells stories. Stories have structure. Structure belongs in the graph.

LORE_SYSTEM.md Section 2.2 specified three relationships for the lore system. Today we verified - and completed - the implementation.

The Audit (#97, #98, #99, #100)

Four issues. Four schema requirements. The surprising discovery: three were already implemented.

  • #97: LoreEpisode node type - already in ontology.py
  • #98: TOLD_TO relationship - already in _save_episode()
  • #99: EXPANDS_UPON relationship - already in _save_episode()
  • #100: SAGA_STARTED_BY relationship - missing

Three out of four. Not bad. But "almost complete" isn't complete.

The Existing Implementation

The _save_episode method already created two relationships:

query = """
CREATE (le:LoreEpisode {
    uuid: $uuid,
    saga_id: $saga_id,
    ...
})
WITH le
MATCH (p:Person {uuid: $captain_uuid})
CREATE (le)-[:TOLD_TO {created_at: $now_ms}]->(p)
WITH le
OPTIONAL MATCH (prev:LoreEpisode {saga_id: $saga_id, chapter: $prev_chapter})
FOREACH (_ IN CASE WHEN prev IS NOT NULL THEN [1] ELSE [] END |
    CREATE (le)-[:EXPANDS_UPON {created_at: $now_ms}]->(prev)
)
"""

TOLD_TO links every episode to its Captain. EXPANDS_UPON chains chapters 2→1, 3→2, etc.

The Missing Piece

SAGA_STARTED_BY marks the first chapter as the saga's origin point. Per the spec:

// Mark saga initiator (first episode only)
(LoreEpisode {chapter: 1})-[:SAGA_STARTED_BY {created_at}]->(Person)

Why does this matter? It answers: "Who started this saga?"

The Fix

Add a conditional FOREACH for chapter 1:

WITH le, p
// SAGA_STARTED_BY: first chapter marks the saga initiator (#100)
FOREACH (_ IN CASE WHEN $chapter = 1 THEN [1] ELSE [] END |
    CREATE (le)-[:SAGA_STARTED_BY {created_at: $now_ms}]->(p)
)

When chapter is 1, create the relationship. Otherwise, skip.

The Query

With the relationship in place, we can now query saga origins:

async def get_saga_origins(
    self,
    trace_id: str | None = None,
) -> list[dict[str, Any]]:
    """Get saga origins - who started each saga (#100)."""
    query = """
    MATCH (le:LoreEpisode {chapter: 1})-[:SAGA_STARTED_BY]->(p:Person)
    WHERE le.saga_id IS NOT NULL
    RETURN le.saga_id as saga_id, le.saga_name as saga_name,
           p.uuid as captain_uuid, p.name as captain_name,
           le.told_at as started_at, le.channel as started_channel
    ORDER BY le.told_at DESC
    """
    ...

Find all chapter 1 episodes with SAGA_STARTED_BY, return the initiator info.
