Running on Autopilot

The Purser keeps your external services in sync. But you shouldn't have to remember to run it.

Now you don't.

APScheduler Integration

Klabautermann already uses APScheduler for periodic jobs. The Archivist scans for stale threads. The Scribe generates daily journals. The Cartographer maps communities weekly.

The Purser joins the crew.

The Pattern

All scheduled jobs follow the same pattern:

from apscheduler.triggers.interval import IntervalTrigger

async def purser_job() -> None:
    job_trace_id = str(uuid.uuid4())
    logger.info(
        "[BEACON] Scheduled Purser sync triggered",
        extra={"trace_id": job_trace_id, "agent_name": "purser"},
    )
    try:
        await purser.sync_all(trace_id=job_trace_id)
        logger.info(
            "[BEACON] Purser sync completed successfully",
            extra={"trace_id": job_trace_id, "agent_name": "purser"},
        )
    except Exception as e:
        logger.error(
            f"[STORM] Purser sync failed: {e}",
            extra={"trace_id": job_trace_id, "agent_name": "purser"},
        )
        # Don't re-raise - let scheduler continue

scheduler.add_job(
    purser_job,
    trigger=IntervalTrigger(minutes=15),
    id="purser_sync",
    name="Purser Gmail/Calendar Sync",
    replace_existing=True,
)

Key Design Decisions

Trace IDs

Each job execution gets a unique trace ID. When you grep logs for that ID, you see the entire sync operation from start to finish.

Error Isolation

If a sync fails, it's logged and forgotten. The scheduler keeps running. The next execution tries again in 15 minutes.

No cascading failures. No stuck jobs.

Coalescing

APScheduler is configured with coalesce=True. If the system was down and missed three scheduled runs, it only runs once when it comes back up.

Configuration

Everything lives in config/scheduler.yaml:

# Purser Configuration
# Synchronizes Gmail and Calendar via delta-sync (#55)
purser:
  enabled: true
  interval_minutes: 15  # Sync every 15 minutes

Set enabled: false to disable. Set interval_minutes: 30 for half-hour syncs. Whatever works for your workflow.

The Full Scheduler Lineup

The scheduler now runs four agents:

  • Syncer: Every 15 minutes - imports from Google Workspace
  • Purser: Every 15 minutes - delta-sync Gmail and Calendar
  • Cartographer: Weekly (Sunday midnight) - detects communities
  • Scribe: Daily (midnight) - generates journal reflections
